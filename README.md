Скрипт подготавливает датасет ru-go-emotions для обучения моделей.

Что делает:
загружает сырые данные из ru-go-emotions-raw.csv;
преобразует текстовые метки эмоций в числовые индексы (с помощью id_to_label.yaml);
формирует колонку labels со списком индексов (например, [1 3 5]);
оставляет только нужные колонки: ru_text, text, labels, id;
сохраняет результат в ru-go-emotions-raw-train.csv;
перед перезаписью проверяет, существует ли файл, и запрашивает подтверждение.

Итог: данные приведены к унифицированному формату (совместимому с simplified), что упрощает их использование в моделях и анализе.
