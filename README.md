Описание сценария предварительной обработки данных (preprocess_data.py)

Этот скрипт служит для предварительной обработки исходных данных из датасета ru-go-emotions и подготовки их к использованию в модели. Основные функции:

Загрузка исходных данных (ru-go-emotions-raw.csv)

Обработка меток эмоций:
— загрузка карты id_to_label.yaml
— преобразование меток в числовой формат
— создание новой колонки labels в виде строки с индексами эмоций, например [1 3 5]

Форматирование данных:
— сохранение только необходимых колонок (ru_text, text, labels, id)
— сохранение итогового файла ru-go-emotions-raw-train.csv

Проверка существования файла перед перезаписью: при наличии файла запрашивается подтверждение (y/n), что предотвращает случайную потерю данных

Особенность
Теперь данные, подготовленные этим сценарием, приведены к формату, совместимому с форматом simplified, что обеспечивает унифицированность и удобство при дальнейшем использовании — например, при обучении модели или анализе.
